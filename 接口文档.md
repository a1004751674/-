# 1.上传新闻信息接口

## 接口描述

上传前端新闻所需的数据

## 请求说明

### HTTP方法 ：`GET`

### 请求URL：`http://127.0.0.1:8000/polls/GetMedia/`

### 请求参数：

无
## 返回说明

### 返回参数

| 字段            | 是否必选 | 类型     | 说明         |
| ---------------| -------- | -------- | ------------ |
| media          | 是       | string   |  媒体方       | 
| title          | 是       | string   | 新闻标题      |
| content        | 是       | string   | 新闻内容      |
| time           | 是       | string   | 发布时间      |
| MediaScr       | 否       | string   | 新闻图片      |

# 3.上传新闻标题热词

## 接口描述

上传新闻标题的词频频率大于20的词语

## 请求说明

### HTTP方法 ：`GET`

### 请求URL：`http://127.0.0.1:8000/polls/GetfrequencyTitle/`

### 请求参数：

无

## 返回说明

### 返回参数

| 字段            | 是否必选 | 类型     | 说明         |
| ---------------| -------- | -------- | ------------ |
| word           | 是       | string   |  词语        | 
| frequency      | 是       | int      | 词频         |
